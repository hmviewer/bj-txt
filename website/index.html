Windows/macOS 데스크톱 로그 분석 도구

후원 로그에서 필요한 BJ 언급만 빠르게 정리하세요.

TXT 파일을 불러오고 `달리, ㄷㄹ`처럼 이름과 초성을 입력하면 메시지 영역만 기준으로 결과를 시간순 정리합니다. 전체 복사, TXT 저장, CSV 내보내기까지 모두 로컬에서 처리합니다.

Features

매일 쓰는 로그 정리 흐름에 맞춘 기능

01

메시지 기준 검색

[DL] 태그, 닉네임, 금액 때문에 오탐이 생기지 않도록 기본 검색은 따옴표 안 메시지만 대상으로 합니다.

02

정확한 초성 매칭

완성형 한글을 초성 배열로 변환해 `ㄷㄹ` 같은 검색어를 안정적으로 처리하고, 쌍자음도 구분합니다.

03

원문 보존 내보내기

복사와 TXT 저장은 화면용 번호나 하이라이트 없이 검색된 원문 로그 라인만 그대로 사용합니다.

04

Excel 친화 CSV

CSV는 UTF-8 BOM을 포함해 한글 깨짐을 줄이고, 시간/후원자/금액/시그/메시지/매칭검색어/원문을 분리합니다.

Security

로그는 내 PC 안에서만 처리됩니다.

로그인, 계정, 서버 업로드 없이 Electron 앱 내부에서 파일을 읽고 분석합니다. 웹사이트는 제품 소개와 다운로드 안내만 제공하며, 로그 파일을 받는 API를 만들지 않았습니다.

  • 로그 파일을 웹사이트에 업로드하지 않습니다
  • 분석 결과를 서버에 저장하지 않습니다
  • 카메라, 마이크, 위치 권한을 요청하지 않습니다
  • 다른 사이트가 이 페이지를 끼워 넣지 못하게 막습니다

Download

Windows와 macOS용 설치 파일을 GitHub Release로 제공합니다.

Windows는 x64 설치 파일, macOS는 Apple Silicon용 DMG/ZIP 파일을 제공합니다. macOS 배포본은 ad-hoc 서명을 적용했으며, Apple Developer 공증 전 파일입니다.
